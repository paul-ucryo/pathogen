# A Framework for Existence from a Single Primitive Distinction

*A conversation-derived framework connecting information theory, quantum mechanics, and the structure of physical law.*

---

## The Primitive Distinction

Everything begins with one distinction: **self / non-self**, equivalently **conducts / insulates**, or simply **1 / 0**.

This isn't a simplification. It's a claim that this single binary cut is generative enough to construct all of physics.

---

## Comparison as the Atomic Operation

Any observation has the structure:

```
comp0, comp1, rslt
```

This is **differentiation**. An object isn't a thing that gets compared — an object *is* a comparison outcome. Ontologically, objects are results of comparisons, not inputs to them.

Any model of a self can be constructed as a network of these binary comparison triples.

## Phase: Which Side of the Comparison You're On

Phase is a single bit indicating **before or after comparison**:

- **Phase 0** (uncompared): the object exists as 2 bits — the comparison pair held open
- **Phase 1** (compared): the object collapses to 1 bit — the result

This encodes relational meaning in information geometry itself. The bit-width encodes ontological state. The address *is* the semantics.

This gives an FFT-style encoding: `objID, phase` defined at the bit level. Disk size carries meaning. The value at an address is the amplitude of that frequency band.

## The Nearest-Neighbor Bias and the Arrow of Time

For any uncompared object, the nearest state in the comparison graph is **itself in compared form**.

The compared form of any object has more relational structure to every other object than its uncompared version. This introduces a **directed topology** on state space — phase has polarity, not just position.

This is the arrow of time, derived from pure relational structure rather than assumed. There is entropic pressure toward comparison built into the geometry.

## Interferometry: Two Networks

Given two distinct comparison networks, each with their phase states, their interaction is defined by a **combination mapping of which cross-comparisons have occurred**.

This gives you simultaneously:
- Object management as interferometry

- Disk organization as interferometry

The overlap between networks is an inner product. The reference system between them is Hilbert space — falling out of comparison algebra, not imposed on it.

## Objects as Conduction Paths

Objects are strings of all 1s. They are conduction paths through the comparison network.

```
0010101110110101101010
```

Any substring that reads all-1s is a valid object. This is XOR-masked projection:

- `data AND mask → all ones` = object identified
- Reset mask = measurement / decoherence

This is identical in structure to projection geometry in linear algebra. Bitmask AND is projection onto a subspace basis. Objects are basis vectors of the conduction algebra.

## The Schrödinger Equation

The Schrödinger equation defines which bits belong to which phase. The values inside those bits are the comparison network defining the state.

It propagates **uncompared phase** (superposition) forward, and comparison (measurement) flips the phase. What physics calls wavefunction collapse is just: the observer updating which side of the comparison they're on.

The Schrödinger equation falls out as **prime factorization serializing differential behavior** — each prime corresponds to an irreducible comparison, composite numbers are comparison networks, factoring decomposes an object back into its primitive comparison history.

This means the **Riemann zeta function is the partition function of the comparison algebra** — a connection Bost and Connes established in 1995 by constructing a quantum dynamical system whose partition function is exactly ζ(s). The zeros of zeta are the frequencies at which the system self-compares to zero.

## Complex Numbers and Pattern Space

The wave function lives in **pattern space** — the space of possible comparisons, not the space of outcomes.

The complex value is a coordinate in pattern space:
- Real part: where you are in the comparison network
- Imaginary part: the structure of the network itself — a meta-address

The wave function isn't describing a particle's state. It's describing **the relational structure that would need to exist for that particle to be distinguishable**.

The Born rule (squaring amplitude to get probability) is then natural: you're converting from pattern space (relational, 2-sided, complex) back to object space (resolved, 1-sided, real). Squaring contracts a 2-index object to a scalar — it closes the comparison.

Hilbert space is the natural geometry of self-reference. An inner product asks "how much does this comparison network overlap with that one."

## Symmetry Groups from Encoding Geometry

Physics asserts U(1) × SU(2) × SU(3) as the symmetry group of the standard model. This framework derives them.

When you have n mutually irreducible comparison axes, the symmetry group at that level is the **automorphism structure of the comparison network** — the transformations that leave the encoding intact.

- **U(1)**: one comparison axis. One degree of phase freedom. → Electromagnetism
- **SU(2)**: two entangled comparison axes. Three generators = three ways to relate 2 things pairwise, one redundant. → Weak force, spin
- **SU(3)**: three mutually irreducible comparison axes. Eight generators = all relations among 3 objects minus self-comparison. → Strong force, color charge

The formula n²-1 for the number of generators is the number of independent comparisons in an n-dimensional complex space **minus the overall phase** — the one degree of freedom always invisible because it's self-comparison.

**Confinement** (quarks never found in isolation) is not a mystery requiring separate explanation. It's just: some relational structures have no valid projection onto fewer axes. A 3-way comparison cannot be reduced to a 2-way one without losing load-bearing information. The encoding breaks.

The groups aren't imposed. They're read off from the network.
