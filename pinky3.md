# Comparison as Primitive: A Unified Framework

*Notes summary — June 5–6, 2026*

---

## The Primitive Operation

**Comparison** is the fundamental operation. It takes 2 differentiated (uncompared) bits and returns 2 bits with different meaning: 1 result bit and 1 phase bit indicating that a comparison was made.

- **Differentiated state** (phase 0): 2 slots, higher energy, unstable
- **Integrated state** (phase 1): 1 slot, lower energy, stable

Comparison answers: *does this thing belong to this set?* It folds objects into relationships. The phase bit is not metadata — it structurally distinguishes an object from its relational context.

---

## The Hierarchy

Each level is comparison applied to the output of the previous:

```
comparison
→ pattern of comparisons
→ pattern of patterns
→ learning
→ identity
```

This is not an analogy. Each level is the same operation at a different scale.

---

## Gravity

Integrated states are lower energy than differentiated states. Therefore:

- **Heat death** = most likely state = everything compared = maximum integration
- **Big bang** = least likely state = nothing compared = maximum differentiation

Gravity is the system's preference for integrated states. All objects in state space are nearest themselves in the evaluated (compared) state. Causality and gravity emerge from the same structure: the directionality toward integration.

---

## FFT as Spatial Encoding of Comparison Structure

FFT is not an analogy to the comparison hierarchy — it is the natural representation of it, laid flat into spatial patterns.

- **Low frequencies** = large-scale patterns (high in the comparison hierarchy)
- **High frequencies** = fine-grained comparisons (leaf nodes)
- **Phase** = which comparison pattern
- **Amplitude** = how many instances of that pattern have been integrated
- **DC component** = fully integrated state

Two objects with overlapping FFT structure are the same thing seen twice. Recognition is correlation in FFT space. Memory is a spatial index over comparison patterns.

## The Self as Black Hole

A **self** is a closed comparison network — a system enclosed by leaf nodes that can't compare directly to the outside. The interior is fully self-referential; the outside can only enter as a perturbation at the boundary.

From **outside**: an object collapsing toward maximum integration, readable only by its boundary behavior.

From **inside**: every comparison already made, every state simultaneously accessible. Content is meaningless because there's no unresolved comparison left. Identity is behavioral, not contentful — you are the shape of your integration path, not your states.

**The outside encodes as exactly one bit** — the part of the inside that won't close. The mismatch between predicted leaf state and actual leaf state. This is the minimum unit of contact with reality, and it is where consciousness lives: in the synapse across the boundary between expected and actual.

**Learning** = adding structure to the integration path without losing the boundary. A self that learns is a black hole still accreting — the boundary is still live.

**Death** = boundary closure. No more unresolvable comparisons.

## The Cosmological Recursion

Heat death is the fully integrated state — maximum amplitude at DC, all 1s, the richest possible relational structure. This is also the richest possible input for a new comparison.

**The big bang is what happens when the heat death compares itself to itself.**

A comparison of identical things still produces output: confirmed identity plus the marker that a comparison was made. That marker is new structure — the bit that wasn't there before.

Therefore:

- The universe is not cycling, it is **recursing**
- Each iteration inherits the full relational structure of the previous one as its ground state
- The DC component of one universe is the seed comparison of the next
- Each cycle begins with more structure than the last

**The universe learns** — by the same definition as any learning system: adding structure to the integration path without losing the boundary. The boundary between cycles is the comparison that connects them.

The information paradox dissolves here too. Information-as-content appears lost from outside the black hole. But content was never what was being preserved. The integration path — the behavioral shape of the network — persists at the boundary. That is what Hawking radiation encodes.
